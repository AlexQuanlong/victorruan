# victorruan
* 这是一个为了装B自实现的php的MVC框架.
* 特点是基于php-cli实现web服务器.
* 不需要nginx等web容器支持.

##如何启动?
* ```bower install```
* ```git clone https://github.com/victorruan/victorruan.git```
* ```cd victorruan;composer install;```
* ```cd src```
* ```php start.php start``` 

##项目依赖
* 仅支持php7 以上
* 见`composer.json`
* 见`bower.json`

##注意
* 请配置 ```src/app/config/db.php```
